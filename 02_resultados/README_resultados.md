# Resultados

Guardar aquí las gráficas exportadas desde MATLAB:

- `graficas_error/`: q deseado vs q real, errores articulares y cartesianos.
- `graficas_torque/`: torques por articulación para cada controlador.
- `trayectoria_obstaculos/`: mapa con obstáculos, ruta A* y trayectoria seguida por el efector.
- `tabla_comparativa.xlsx`: plantilla para consolidar métricas.
